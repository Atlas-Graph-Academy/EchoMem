# Memory Signal

Extract emotional signals and patterns from memory collections. The sensing layer that powers emotion-aware search — transforming raw memories into structured emotional fingerprints for matching, clustering, and discovery.

## Usage

```bash
memory-signal analyze <user-id>    # Extract signals from all memories
memory-signal compare <id-a> <id-b> # Compare emotional fingerprints
memory-signal trends <user-id>     # Emotional pattern over time
```

## Features

- Multi-dimensional emotional encoding (beyond sentiment polarity)
- Temporal pattern detection (recurring themes, emotional arcs)
- Signal strength scoring for matching relevance
- Privacy-preserving: signals are derived, not raw content
