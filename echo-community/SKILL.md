# Echo Community

Build and nurture micro-communities around shared memories. Automatically discover groups of people with resonant experiences, facilitate introductions, and grow organic communities rooted in genuine understanding — not algorithms.

## Usage

```bash
echo-community discover <topic>     # Find communities around a theme
echo-community create <name>        # Seed a new community from matched users
echo-community insights <group-id>  # Shared patterns and narratives
```

## Features

- Memory-based community formation (shared experiences, not interests)
- Organic growth through resonance matching
- Community narratives auto-generated from collective memories
- Trust scoring based on memory depth and authenticity
- Privacy-first: opt-in only, no data leaks
